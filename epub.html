HTML Folder → Text PDF

Extracts real, selectable text straight from your HTML chapter files and lays it out as an actual text-based PDF — no screenshots, no rasterizing. The result stays small and stays sharp at any zoom level. Headings, bold, and italics are preserved; exact pixel alignment is not the goal here.

Select your chapter folder

Choose the folder containing your .html / .xhtml files. They're read in natural sort order (chapter_01, chapter_02...).

Build the PDF

Each file becomes a new chapter, starting on a fresh page, in the order shown above.

Runs entirely in your browser — nothing is uploaded anywhere. Built with jsPDF.
